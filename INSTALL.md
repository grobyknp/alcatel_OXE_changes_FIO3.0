Инструкция: подготовка окружения и запуск changes_FIO3.0.py

## Шаг 0. Установка Python (если ещё не установлен)
Откройте командную строку от имени администратора.

Введите python --version.
Самопроверка: Если вы видите версию (например, Python 3.12.2) → Python уже есть, переходите к шагу 1.
Если ошибка («не распознана»), скачайте Python с официального сайта. При установке обязательно отметьте галочку «Add Python to PATH». После установки перезапустите командную строку от администратора и повторите проверку.

## Шаг 1. Создание виртуального окружения, выполняется в командной строке
# 1.1 Перейдите в папку, где лежит ваш скрипт change_FIO3.0.py:

cd C:\путь\к\вашей\папке

### 1.2 Создайте виртуальное окружение (назовём его venv):

python -m venv venv

### 1.3 Самопроверка: После выполнения появилась папка venv в текущей директории. Внутри неё есть подпапки Scripts и Lib.

## Шаг 2. Активация виртуального окружения, выполняется в командной строке
### 2.1 Активируйте окружение (Windows):

venv\Scripts\activate

### 2.2 Самопроверка: В начале командной строки появился текст (venv).
Дополнительная проверка: введите where python — первым в списке должен быть путь C:\...\venv\Scripts\python.exe (внутри вашей папки).

## Шаг 3. Запуск вашего скрипта в виртуальном окружении, выполняется в командной строке
### Убедитесь, что в командной строке всё ещё видно (venv). Затем выполните:

python change_FIO3.0.py
### Самопроверка: Скрипт запустился без ошибок импорта (например, ModuleNotFoundError). Если он что-то выводит или выполняет свою задачу — значит, окружение работает корректно.
