# alcatel_OXE_changes_FIO3.0
# Скрипт обновления данных абонента на АТС Alcatel OXE

# Alcatel OXE Subscriber Updater

Скрипт для автоматического обновления имени и фамилии абонента на АТС Alcatel OXE через Telnet.
Имеет графический интерфейс (Tkinter), выполняет транслитерацию с кириллицы на латиницу и отправляет команды в консоль управления АТС.

---

## Требования к Python

- Версия Python: 3.9 - 3.13 (включительно)
- Проверено на: Python 3.12.9
- Примечание: в Python 3.13 модуль telnetlib ещё доступен, но в будущих версиях может быть удалён. При необходимости можно заменить на telnetlib3 (не требуется для текущей версии).
- Стандартные библиотеки: все модули входят в стандартную поставку Python, кроме tkinter (требует отдельной установки на некоторых системах).

---

## Установка и настройка окружения

1. Установите Python 3.9-3.13 с официального сайта python.org.
   На Windows обязательно отметьте опцию "Add Python to PATH".

2. Установка tkinter (если не установлен):

   - Windows / macOS: tkinter входит в стандартную сборку.
   - Linux (Debian/Ubuntu):
sudo apt update
sudo apt install python3-tk

- Linux (Red Hat/Fedora):
sudo dnf install python3-tkinter


3. Дополнительные библиотеки не требуются.

## Конфигурация

### Файл servers.json

Скрипт использует файл servers.json для хранения соответствия:
код региона -> IP-адрес АТС и пароль для учётной записи mtcl.

Путь по умолчанию: C:\alcatel_script\servers\servers.json (можно изменить в классе Config).

#### Структура файла:

```json
[
{
"region_code": "01",
"address": "192.168.1.1",
"mtcl": "password_for_mtcl"
},
{
"region_code": "02",
"address": "192.168.2.1",
"mtcl": "another_password"
}
]
```
## Переменная           Описание
region_code	            две цифры кода региона (первые две цифры после 9 в номере)
address	               IP-адрес АТС
mtcl	                  пароль для учётной записи mtcl (логин фиксирован)

Для каждого региона, который встречается в номерах, должна быть запись в файле.

При отсутствии записи скрипт выдаст ошибку.

## Логирование
Логи записываются в файл:
C:\alcatel_script\changes.log (путь настраивается в Config.LOG_FILE).
Директория создаётся автоматически при первом запуске.

Формат номера абонента
Скрипт ожидает номер в формате: 9xxyyyy

xx - код региона (2 цифры)

yyyy - остальные 4 цифры

IP-адрес АТС определяется по коду региона xx из servers.json.

## Запуск
Откройте командную строку (терминал).

Перейдите в папку со скриптом.

Выполните команду:

python change_FIO3.0.py
Откроется графическое окно:

1) Введите номер абонента (например, 9011666).

Автоматически отобразится IP-адрес АТС (из servers.json).

2) Введите фамилию и имя кириллицей.

3) Нажмите "Переименовать".

После выполнения появится сообщение об успехе или ошибке.

## Рекомендации по работе с кодом
Config - хранение настроек (пути, таймауты, константы)

Transliterator - транслитерация кириллицы в латиницу

ServerManager - чтение servers.json, получение IP и пароля по коду региона

AlcatelSession - низкоуровневая работа с Telnet

AlcatelOXEClient - выполнение mgr-скриптов на АТС

SubscriberUpdater - основная бизнес-логика обновления абонента

GUI - Tkinter интерфейс
