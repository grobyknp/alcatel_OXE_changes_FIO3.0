# alcatel_OXE_changes_FIO3.0
# Скрипт обновления данных абонента на АТС Alcatel OXE

Данный скрипт предназначен для автоматического обновления имени и фамилии абонента на АТС **Alcatel OXE** через Telnet.  
Скрипт имеет графический интерфейс (Tkinter), выполняет транслитерацию с кириллицы на латиницу и отправляет команды в консоль управления АТС.

## Требования к Python

- Версия Python: **3.9 – 3.13** (включительно)
- Проверено на Python 3.12.9
- Примечание: в Python 3.13 модуль `telnetlib` всё ещё доступен, но в будущих версиях может быть удалён. При необходимости можно заменить на `telnetlib3` (не требуется для текущей версии).

**Стандартные библиотеки:** все используемые модули входят в стандартную поставку Python, за исключением `tkinter`, который в некоторых системах требует отдельной установки.

## Установка и настройка окружения

1. Установите **Python 3.9–3.13** с официального сайта python.org.  
   При установке на Windows обязательно отметьте опцию *«Add Python to PATH»*.

2. Установка `tkinter` (если не установлен):
   - **Windows / macOS**: `tkinter` входит в стандартную сборку, дополнительных действий не требуется.
   - **Linux (Debian/Ubuntu)**:
     ```bash
     sudo apt update
     sudo apt install python3-tk
Linux (Red Hat/Fedora):

bash
sudo dnf install python3-tkinter
Убедитесь, что все необходимые модули доступны. Дополнительные библиотеки не требуются.

Конфигурация
1. Файл servers.json
Скрипт использует файл servers.json для хранения соответствия кода региона → IP-адрес АТС и пароля для учётной записи mtcl.

Путь к файлу по умолчанию:
C:\alcatel_script\servers\servers.json
(путь можно изменить в классе Config).

Структура файла:

json
[
  {
    "region_code": "01",
    "address": "192.168.1.1",
    "mtcl": "password_for_mtcl"
  },
  {
    "region_code": "02",
    "address": "192.168.2.1",
    "mtcl": "another_password"
  }
]
region_code – две цифры кода региона (первые две цифры после 9 в номере абонента).

address – IP-адрес АТС.

mtcl – пароль для учётной записи mtcl (логин фиксирован).

Важно:

Убедитесь, что для каждого региона, который встречается в номерах, в файле есть соответствующая запись.

При отсутствии записи для кода региона скрипт выдаст ошибку.

2. Логирование
Логи записываются в файл C:\alcatel_script\changes.log (путь настраивается в Config.LOG_FILE).
Директория создаётся автоматически при первом запуске.

3. Формат номера абонента
Скрипт ожидает номера в формате 9xxyyyy, где:

xx – код региона (2 цифры),

yyyy – остальные 4 цифры.

IP-адрес АТС определяется по коду региона xx из файла servers.json.

Запуск
Откройте командную строку (терминал).

Перейдите в папку со скриптом.

Выполните команду:

bash
python change_FIO3.0.py

Откроется графическое окно:

Введите номер абонента (например, 9011666).

Автоматически отобразится IP-адрес АТС (из servers.json).

Введите фамилию и имя кириллицей.

Нажмите «Переименовать».

После выполнения появится сообщение об успехе или ошибке.

Рекомендации по работе с кодом
Структура проекта
Config – хранение настроек (пути, таймауты, константы).

Transliterator – транслитерация кириллицы в латиницу.

ServerManager – чтение файла servers.json, получение IP и пароля по коду региона.

AlcatelSession – низкоуровневая работа с Telnet.

AlcatelOXEClient – выполнение mgr-скриптов на АТС.

SubscriberUpdater – основная бизнес-логика обновления абонента.

GUI – Tkinter интерфейс.
